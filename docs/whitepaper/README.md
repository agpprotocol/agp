Version 0.9.1 — Public Review Draft

- [Read the whitepaper online](./AGP_Whitepaper_v0.9.1.md)

This document is experimental. It is not a production standard and has not received independent security review.

Review is particularly requested on:

- novelty and relationship to existing systems;
- canonicalization and deterministic resolution;
- authority, replacement, and revocation semantics;
- transparency and visibility assumptions;
- benchmark methodology;
- the boundary between AGP and existing policy/workflow components.
