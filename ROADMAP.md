# AGP Roadmap

## Current: experimental validation

- deterministic semantic conformance;
- Python and Go interoperability;
- signed envelopes;
- transparency log;
- AGP vs workflow benchmark.

## Next

- external reproduction by independent developers;
- formal JSON schemas;
- clearer normative specification language;
- third independent implementation;
- real-world pilot integration;
- expanded benchmark against a mature workflow engine;
- independent security review.

## Later, only if adoption evidence exists

- stable versioning policy;
- SDK packaging;
- interoperability registry;
- public test service;
- standards-track exploration.

AGP will not be promoted as a production standard until external validation and
real deployment evidence exist.
